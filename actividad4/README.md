# 📂 Actividad 4: Cálculo de suma y media

## 📝 Descripción

Se implementa un programa que:

- Lee los números del archivo `NumerosReales.txt`
- Calcula la suma total
- Calcula la media aritmética
- Muestra los resultados por pantalla

## ⚙️ Tecnologías usadas

- BufferedReader
- String.split()
- String.valueOf()
- Double.parseDouble()

## 🎯 Objetivo

Trabajar con datos numéricos almacenados en texto y procesarlos en Java.
