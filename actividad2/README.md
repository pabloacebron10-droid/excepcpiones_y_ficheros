# 📂 Actividad 2: Lectura con FileReader

## 📝 Descripción

Se lee el archivo `Main.java` de un proyecto previamente realizado y se muestra su contenido por pantalla.

## ⚙️ Tecnologías usadas

- FileReader
- Manejo de excepciones (try-catch)

## 🎯 Objetivo

Comprender la lectura de archivos carácter a carácter.
